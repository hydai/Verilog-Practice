If you want to run the simulation with golden test case 1 using "make ts1".
If you want to run the simulation with golden test case 2 using "make ts2".
If you want to run the simulation with my test case  using "make my".
If you want to run code checking, using "make check".
After finishing the simulation, you can use "make clean"
